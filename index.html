OTROS MÉRITOS O DATOS DE INTERÉS

Como Proyecto Final de la carrera Ingeniería en Computación realice un desarrollo para la misma facultad; el mismo consistió en un sistema donde los alumnos vía teléfono celular pueden consultar información sobre su estado académico, sus estadísticas, inscribirse en materias para su cursado, inscribirse y borrarse de mesas de exámenes, etc.

El sistema “Wapalumnos” se realizo utilizando diversas tecnologías: WML como lenguaje de programación estático y ASP para la generación dinámica de datos. El sistema gestor de base de datos que se uso fue Microsoft SQL Server 2000. Como servidor Web se utiliza Internet Information Server 6.0 de Microsoft. El proyecto tuvo gran aceptación por parte de Universidad por lo cual el sistema se encuentra actualmente en uso en la Facultad de Ciencias Exactas y Tecnología de la Universidad Nacional de Tucumán.
